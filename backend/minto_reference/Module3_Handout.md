**Версия для сотрудников (памятка)**

**Зачем**

Чтобы вас **сразу понимали** на планёрках и встречах, чтобы
1--2-минутные выступления были **чёткими и уверенными**, а ИИ давал
**структурные черновики** по вашей задаче.

**Пирамида Минто --- устно**

-   **Вершина**: главное предложение (ответ/предложение) --- **сразу**.

-   **Опоры**: 2--3 пункта **одного типа** (причины/выгоды/шаги).

-   **Детали**: 1--2 факта на опору --- только нужное.

**Совещания и планёрки**

-   Откройте **вершиной**: цель/решение (1 фраза).

-   Повестка = **опоры** (2--4 пункта) в одной логике: **хронология /
    структура / сравнение / приоритет**.

-   Внутри пункта --- только факты и решение.

**Краткие презентации (1--3 мин)**

-   **Вершина:** «Результат/предложение».

-   **3 опоры:** напр., «Финансы / Клиент / Операции».

-   **Детали:** по 1--2 факта; финал --- **призыв/следующий шаг**.

**Промпты к нейросети (чтобы получить структурированный черновик)**

1.  **Роль:** «Ты --- аналитик/редактор/PM...».

2.  **Задача:** «Составь план 2-мин доклада по пирамиде Минто...».

3.  **Контекст:** «Вершина: ...; факты: ...».

4.  **Формат:** «список с заголовками / план презентации».

5.  **Тон:** «деловой, краткий».

**Шаблон 2-мин выступления**

**Вершина → 3 опоры → Следующий шаг.** (Однотипные опоры, короткие
детали.)

**Частые ошибки → как исправить**

-   **Нет вершины** → скажите главную мысль **первой фразой**.

-   **Смешаны типы опор** → выберите **один тип** (только причины/только
    выгоды/только шаги).

-   **Слишком много деталей** → оставьте по **1--2 факта** на опору.

**Практика (сдать в этом модуле)**

1.  Репак устного ответа для планёрки (вершина → опоры → просьба).

2.  **Видео 60--120 сек** по шаблону.

3.  Промпт к ИИ для плана доклада (Роль/Задача/Контекст/Формат/Тон) +
    ответ модели.
